## 位运算符

| 运算符 | 描述 | 示例 |
| --- | --- | --- |
| `&` | 位与 | `a & b` |
| `|` | 位或 | `a | b` |
| `^` | 位异或 | `a ^ b` |
| `~` | 位取反 | `~a` |
| `<<` | 左移 | `a << b` |
| `>>` | 右移 | `a >> b`

```csharp
int a = 60;     // 0011 1100
int b = 13;     // 0000 1101
int c = a & b;  // 与运算，结果为 12
int d = a | b;  // 或运算，结果为 61
int e = a ^ b;  // 异或运算，结果为 49
int f = ~a;     // 取反运算，结果为 -61
int g = a << 2; // 左移运算，结果为 240
int h = a >> 1; // 右移运算，结果为 30
```

## C# 中的运算符优先级

- 优先级简易概括：有括号先括号，后乘除在加减，然后位移再关系，逻辑完后条件，最后一个逗号 , 。

### C# 中的运算符优先级是从最高到最低的顺序。

| 类别 | 运算符 | 结合性 |
| --- | --- | --- |
| 后缀 | `() [] -> . ++ --` | 从左到右 |
| 一元 | `+ - ! ~ ++ -- sizeof typeof * &` | 从右到左 |
| 乘除 | `/ * %` | 从左到右 |
| 加减 | `+ -` | 从左到右 |
| 位移 | `<< >>` | 从左到右 |
| 关系 | `< > <= >=` | 从左到右 |
| 相等 | `== !=` | 从左到右 |
| 位与 | `&` | 从左到右 |
| 位异或 | `^` | 从左到右 |
| 位或 | `|` | 从左到右 |
| 逻辑与 | `&&` | 从左到右 |
| 逻辑或 | `||` | 从左到右 |
| 条件 | `? :` | 从右到左 |
| 赋值 | `= += -= *= /= %= &= |= ^= <<= >>=` | 从右到左 |
| 逗号 | `,` | 从左到右 |
