# C# 常量

- 常量是固定值，程序执行期间不会改变。常量可以是任何基本数据类型，比如整数常量、浮点常量、字符常量或者字符串常量，还有枚举常量。

## 整数常量

### 十进制:

- 十进制常量是最常见的整数常量，通常用于表示十进制数字。

```csharp
int i = 100;
```

### 二进制:

- 二进制常量是以 0b 开头的十进制数字，通常用于表示二进制数字。

```csharp
int j = 0b100;
```

### 十六进制:

- 十六进制常量是以 0x 开头的十进制数字，通常用于表示十六进制数字。

```csharp
int k = 0x100;
```

### unsigned:

- unsigned 常量是无符号整数常量，通常用于表示无符号整数，可以用于表示非负整数。

```csharp
int l = 100u;
```

### long:

- long 常量是长整数常量，通常用于表示长整数，可以用于表示大整数。

```csharp
long m = 100L;
```

## 浮点常量

### 单精度浮点常量:

- float 常量是单精度浮点常量，通常用于表示单精度浮点数，可以用于表示小数。

```csharp
float f = 3.14f;
```

### 双精度浮点常量:

- double 常量是双精度浮点常量，通常用于表示双精度浮点数，可以用于表示大数。

```csharp
double d = 3.14;
```

## 字符常量

- 字符常量是单个字符，通常用于表示字符。

- 转义序列码：

| 转义序列 | 描述 |
| --- | --- |
| \' | 单引号 |
| \" | 双引号 |
| \\ | 反斜线 |
| \a | 响铃 |
| \b | 退格 |
| \f | 换页 |
| \n | 换行 |
| \r | 回车 |
| \t | 制表符 |
| \v | 垂直制表符 |

## 字符串常量

- 字符串常量是括在双引号 "" 里，或者是括在 @"" 里。

```csharp
string str = "Hello, World!";
string str2 = @"C:\Program Files";
```

## 定义常量

- 常量是使用 const 关键字来定义的 。定义一个常量的语法如下：

```csharp
const <data_type> <constant_name> = value;
```
