# C# 判断

## if 语句

- if 语句格式：if (条件表达式) { 语句 }

```csharp
if (condition)
{
    // 语句
}
```

## if...else 语句

- if...else 语句格式：if (条件表达式) { 语句 } else { 语句 }

```csharp
if (condition)
{
    // 语句
}
else
{
    // 语句
}
```

## if...else if...else 语句

- if...else if...else 语句格式：if (条件表达式1) { 语句1 } else if (条件表达式2) { 语句2 } else { 语句3 }

```csharp
if (condition1)
{
    // 语句1
}
else if (condition2)
{
    // 语句2
}
else
{
    // 语句3
}
```

## switch 语句

- switch 语句格式：switch (表达式) { case 值1: 语句1; break; case 值2: 语句2; break; default: 语句3; }

```csharp
switch (expression)
{
    case value1:
        // 语句1
        break;
    case value2:
        // 语句2
        break;
    default:
        // 语句3
        break;
}
```

## ? : 运算符

- ? : 运算符格式：表达式1 ? 表达式2 : 表达式3

```csharp
expression1 ? expression2 : expression3;
```


# C# 循环

## while 语句

- while 语句格式：while (条件表达式) { 语句 }

```csharp
while (condition)
{
    // 语句
}
```

## do...while 语句

- do...while 语句格式：do { 语句 } while (条件表达式)

```csharp
do
{
    // 语句
} while (condition);
```

## for 语句

- for 语句格式：for (初始化语句; 条件表达式; 迭代语句) { 语句 }

```csharp
for (初始化语句; 条件表达式; 迭代语句)
{
    // 语句
}
```

## 循环控制语句

- break 语句用于跳出循环语句，continue 语句用于跳出当前循环语句的迭代部分，goto 语句用于跳转到指定的标签
