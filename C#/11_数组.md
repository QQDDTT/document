# C# 数组（Array）

- 数组是一个存储相同类型元素的固定大小的顺序集合。

## 声明数组

```csharp
<data_type> [] <variable_name>
```

- data_type:数据类型
- variable_name:变量名

## 初始化数组

```csharp
int [] arr = new int [5];
int [] arr = {10, 20, 30, 40, 50};
int [] arr = new int [5] {10, 20, 30, 40, 50};
```

## 访问数组元素

```csharp
int [] arr = {10, 20, 30, 40, 50};

// 访问第一个元素
Console.WriteLine(arr[0]);

// 访问最后一个元素
Console.WriteLine(arr[arr.Length - 1]);
```

## 使用 foreach 循环

```csharp
int [] arr = {10, 20, 30, 40, 50};

foreach (int i in arr)
{
    Console.WriteLine(i);
}
```
