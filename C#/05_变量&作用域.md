# C# 变量

## C# 中提供的基本的值类型大致可以分为以下几类：

- 整数类型: byte, sbyte, short, ushort, int, uint, long, ulong
- 浮点类型: float, double
- 布尔类型: bool
- 字符串类型: string
- 空类型: null

- C# 4.0引入了动态类型 (dynamic),它允许在运行时推断变量的类型。这在一些特殊情况下很有用，但通常最好使用静态类型以获得更好的性能和编译时类型检查。

```csharp
dynamic d = 100;
```

## C# 中变量定义的语法：

```csharp
<data_type> <variable_list>;
```

```csharp
int i, j, k;
char c, ch;
float f, salary;
double d;
```

## 变量的命名规则

- 变量名可以包含字母、数字和下划线。
- 变量名必须以字母或下划线开头。
- 变量名区分大小写。
- 避免使用 C# 的关键字作为变量名。

## C# 中的变量初始化

```csharp
<data_type> <variable_name> = value;
```

```csharp
int d = 3, f = 5;    /* 初始化 d 和 f. */
byte z = 22;         /* 初始化 z. */
double pi = 3.14159; /* 声明 pi 的近似值 */
char x = 'x';        /* 变量 x 的值为 'x' */
```

# C# 变量作用域

- 在 C# 中，变量的作用域定义了变量的可见性和生命周期。

## 局部变量

- 在方法、循环、条件语句等代码块内声明的变量是局部变量，它们只在声明它们的代码块中可见。

## 块级作用域

- 在 C# 7及更高版本中，引入了块级作用域，即使用大括号 {} 创建的任何块都可以定义变量的作用域。

## 方法参数作用域

- 在 C# 中，方法参数是方法的局部变量，它们只在方法内可见。

## 全局变量

- 在类的成员级别定义的变量是成员变量，它们在整个类中可见，如果在命名空间级别定义，那么它们在整个命名空间中可见。
