# Rust 环境搭建

## Linux / macOS

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

## Windows

- 下载 rustup-init.exe 可执行文件。

```url
https://static.rust-lang.org/rustup/dist/i686-pc-windows-gnu/rustup-init.exe
```

## 完成了 Rust 的安装，可以通过以下命令测试

```bash
rustc -V        # 注意的大写的 V
```
